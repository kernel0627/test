# 2026 年北京高校数学建模校际联赛论文格式规范

1. 每个参赛队伍应提交的文件分为两个部分：一个是论文（PDF），一个支撑文件（压缩），如果没有用到程序代码或其他要求，可不提供支撑文件。

2. 论文 A4 版面, 上下左右各留出至少 2.5 厘米的页边距。文件格式为PDF；文件大小不超过 20MB。注意参赛论文电子版文件不要压缩。

3. 论文第一页为论文标题、摘要和关键词（无需译成英文），并从此页开始编写页码；页码必须位于每页页脚中部，用阿拉伯数字从“1”开始连续编号，不要页眉。注意：摘要应该是一份简明扼要的详细摘要，请认真书写（但篇幅不能超过一页）。

4. 论文从第二页开始是论文正文；正文之后是参考文献和论文附录（页数不限）。

5. 论文附录内容应包括支撑材料的文件列表，应提供参赛者实际使用的软件名称、建模所用到的全部完整、可运行的源程序代码（含 EXCEL、SPSS 等软件的交互命令）等。如果缺少必要的源程序、程序不能运行或运行结果与论文不符，都可能会被取消评奖资格。如果确实没有用到程序，应在论文附录中明确说明“本论文没有用到程序”。赛题中提供的原始数据不必写入附录。

5. 支撑材料内容包括用于支撑模型、结果、结论的所有必要材料，至少应包含建模所用到的所有可运行源程序、自主查阅使用的数据资料（赛题中提供的原始数据除外）、较大篇幅中间结果的图表等。将所有支撑材料文件使用 WinRAR 软件压缩在一个文件中（后缀为 RAR 或 ZIP，大小不超过 20MB）。支撑材料的文件列表应放入论文附录；如果确实没有需要提供的支撑材料，可以不提供支撑材料文件，并在论文附录中注明“本论文没有支撑材料”。如果支撑材料文件与论文内容不相符，该论文可能会被取消评奖资格。所有文件中不能有显示参赛者身份和所在学校的信息, 特别注意: 支撑文件压缩包名字, 压缩包内各子文件的文件名都不允许出现学校或参赛者的任何信息。

# 6. 论文和附录中任何地方都不能有显示参赛者身份和所在学校的信息。

7. 引用别人的成果或其他公开的资料(包括网上资料)必须按下列方式在正文引用处和参考文献中明确列出。正文引用处用方括号标示参考文献的编号，如[1]、[3]等；引用书籍必须指出页码。参考文献按正文中的引用次序列出，其中书籍的表述方式为：

[编号] 作者，书名，出版地：出版社，出版年。

参考文献中期刊杂志论文的表述方式为：

[编号] 作者，论文名，杂志名，卷期号：起止页码，出版年。

参考文献中网上资源的表述方式为：

[编号] 作者，资源标题，网址，访问时间（年月日）。

# 不符合本格式规范的论文将被视为违反竞赛规则，可能被取消评奖资格。
