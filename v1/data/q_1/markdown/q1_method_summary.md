# 第一问方法说明

## 维护口径

附件 2 只记录中维护和大维护，因此本文只将中维护、大维护作为显式维护事件。题面中的小维护约每 3 至 5 天进行一次，但没有逐次日期记录，本文不估计 `G_small` 或 `rho_small`，而是将小维护作为背景维护机制。

## 缺失口径

维护日整天无数据定义为 `maintenance_gap`，属于结构性缺失；非维护日整天无数据定义为 `random_gap`。维护性缺失不插值，不进入维护效果、回归和周期衰减计算。

## 指标口径

`r_i*` 表示小维护背景下，相邻显式维护之间的净衰减率。维护即时增益 `G` 用于描述恢复幅度，恢复比例 `rho` 用于第二问状态转移建模。

## 大维护口径

大维护每台设备每年允许 0 至 4 次，不是年度必做维护。由于大维护样本少且不均衡，设备级 `rho_major` 在第二问中需要向全局参数收缩。
