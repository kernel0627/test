# 第一问结果解释

## 核心规律

透水率在小维护持续存在的背景下，受到中维护和大维护等显式维护事件影响。中维护相对频繁，大维护是低频重维护，可能某些年份为 0 次。

## 维护周期

本文的维护周期是相邻有记录的中维护或大维护之间的周期，不是大维护年度周期。周期衰减率解释为小维护背景下的净衰减。

## 恢复比例

事件级 `rho_raw` 允许小于 0 或大于 1。第一问统计使用 `rho_clipped`，第二问使用设备级收缩后的 `rho_medium_use` 和 `rho_major_use`。

## 大维护结论

大维护效果估计的不确定性高于中维护。论文中只说明现有样本下大维护即时恢复优势不明显或需谨慎解释，不写成“大维护一定更好”或“大维护没用”。
